
# Cosmic Explorer

A space exploration web application that immerses users in the latest discoveries and challenges of the cosmos. This web app informs, engages, and entertains through educational articles, interactive quizzes, and space-themed games.

## Features

- **Articles Section**: 10 curated space science and exploration articles covering recent breakthroughs, missions, and discoveries.
- **Interactive Quizzes**: Each article ends with a short quiz to test comprehension.
- **Space Games**: Three browser-based games inspired by space exploration:
  - Asteroid Defender: Protect Earth from incoming asteroids in this arcade-style game
  - Space Trivia Challenge: Test your knowledge with questions from articles and general astronomy
  - Planet Matcher: A memory matching game with a space theme

## Technologies Used

- HTML
- CSS
- JavaScript

## Setup and Installation

1. Clone the repository
2. Open index.html in your web browser
3. No additional installation required
